# Decode Morse Code
This project decodes Morse code messages using Ruby.

## Learning Objectives
- Run Ruby programs using the command line
- Use Ruby syntax for basic operations
- Solve coding challenges using Ruby

## Features
- Decode a single Morse character
- Decode a Morse word
- Decode a full Morse message

## Example
Input:
-- -.--   -. .- -- .

Output:
MY NAME
